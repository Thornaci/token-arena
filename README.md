# Token Arena

**See what the model sees.** A gamified, open-source, fully static web app that teaches software developers how LLMs actually work — tokenization, the stateless request loop, the context window, instruction hierarchy, context rot, and more. Real mechanics under a game, in the spirit of [Oh My Git!](https://ohmygit.org).

> Status: early work in progress (Phase 1 foundation).

## Development

```sh
npm install
npm run dev      # start dev server
npm test         # run unit tests
npm run build    # static production build
npm run preview  # preview the production build
```
